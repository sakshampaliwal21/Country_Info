# Country Explorer

This project is a simple web application that displays information about countries using a public API. It shows details like country name, capital, population, region, flag, and currency.

Users can search countries by name, filter them by region, and sort them based on population. It also includes features like dark mode and a favorite button for interaction.

The data is fetched using the Fetch API and displayed dynamically on the webpage.

I have deployed this project by sharing it through WhatsApp for easy access and testing.

This project helped me understand API integration, JavaScript functions, and UI design.
