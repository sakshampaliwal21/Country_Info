# Country_Info

## Overview

Country Explorer is a simple and interactive web application that allows users to explore information about countries around the world. It uses a public API to fetch real-time data and displays it in a clean and user-friendly interface.

---

## Features

* Search countries by name
* Filter countries by region
* Sort countries by population or name
* Display country flags
* Fully responsive design
* Fast data fetching using API

---

## API Used

This project uses the REST Countries API:

https://restcountries.com/v3.1/all?fields=name,capital,currencies,population,flags,region

### Data Provided:

* Country Name
* Capital
* Population
* Currency
* Region
* Flag
